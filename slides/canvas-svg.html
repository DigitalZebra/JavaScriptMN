Overview

What?

Canvas
  • New HTML tag part of HTML 5 specification
  • Provides low level raster based graphics API
SVG
  • Scalable Vector Graphics
  • XML file format which defines vector graphics
  • <SVG> tag can be embedded in HTML and rendered by browsers

Why Canvas and SVG?

  • No Flash in the browser on mobile devices!

Canvas

Quick facts:

  • Introduced by Apple in Webkit circa 2004
  • Part of the HTML 5 working draft
  • Supported by the following major browsers:
    Chrome 4+, FF 2+, IE 9+, Opera 9+, Safari 3.1+, iOS Safari 3.2+, Android Browser 2.1+, Opera Mobile 10.0+
  • Looks like: <canvas>…</canvas>

Canvas

  • Provides a low level raster based drawing API, meaning that you can manipulate the screen on a pixel by pixel basis.
  • Does NOT provide a scene graph. The canvas element has no memory of what was rendered previously, you have to track state yourself.
  • Provides methods for rendering images, gradients, and shadows, along with geometric primitives.

Canvas

The <canvas> element provides a getContext(string) method which fetches the drawing context to be used.
This allows the browser to support multiple drawing APIs or specifications.

Currently, two specifications exist:

2d
The W3C standard and most widely supported context
Provides a 2-D drawing API 
var context = canvasDomElement.getContext("2d");
WebGL
Experimental, only supported in WebKit and FireFox
Provides a 3-D draw API which mimics the OpenGL ES API 
var context = canvasDomElement.getContext("webgl");

Canvas

Code Sample!

		<body>
			<canvas id="myCanvas" width="500" height="500"></canvas>
			<script>
			var can = document.getElementById('myCanvas');
			var ctx = can.getContext('2d');
		
			ctx.fillStyle = "#006600";
			ctx.beginPath();
			ctx.arc(200, 200, 50, 0, Math.PI * 2); // draw circle
			ctx.fill();
			</script>
		</body>

Canvas

Code Sample!

This draws the transparent Wikipedia logo on the canvas. 

		<body>
			<canvas id="myCanvas" width="500" height="500"></canvas>
			<script>
			var can = document.getElementById('myCanvas');
			var ctx = can.getContext('2d');
		
			var imageObj = new Image();
		
  			function drawPattern() {
				ctx.drawImage(imageObj, 0, 0);
  			}
  			
  			imageObj.onload = drawPattern;
  			imageObj.src = "http://upload.wikimedia.org/wikipedia/en/b/bc/Wiki.png";
			</script>
		</body>
